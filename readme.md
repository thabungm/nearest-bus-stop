# Find Nearest Bus Stops

### Features implemented
    - Seed data for bus stop locations
    - Create bus 
    - Form validation for bus creation
    - Bus Listing
    - Edit Bus
    - Get nearest bus-stops
    - View buses in a bus stop
    - Switch my location (hardcoded) by clicking buttons
